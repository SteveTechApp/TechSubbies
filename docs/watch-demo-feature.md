# Watch a Demo feature

The Watch a Demo page lets a visitor select one of three customer types and walk through a guided POV slideshow.

## Customer types

- Engineer / subcontractor
- Hiring company / project owner
- Resourcing company / company account

## Purpose

The feature explains the TechSubbies process using page-style demo screens, narration, user actions, platform responses and expected results.
