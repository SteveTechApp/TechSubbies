# TechSubbies Live Opportunity Intake Builder

## Purpose

The Live Opportunity Intake Builder turns the Matching Engine from a static demo into a usable workflow.

A company or resourcing user can define:

- AV / IT / AV+IT market
- role type
- must-have skills
- nice-to-have skills
- required product/platform tags
- minimum product experience level
- location mode

The page then runs the matching engine and returns ranked candidate results.

## Match output

Each candidate result includes:

- GOOD MATCH / PARTIAL MATCH / NO MATCH
- overall score
- role-fit score
- must-have score
- product/platform score
- reasons
- risks/gaps
- next qualification questions

## Scope boundary

This builder is skills and product-knowledge based only.

It does not decide:

- insurance suitability
- documents
- tools/testers
- pay rate
- contract terms
- legal eligibility
- final availability

Those belong in separate profile and availability areas.
